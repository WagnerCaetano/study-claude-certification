## 🎯 How to Use This Study Guide

1. **Follow the flow** – Start with Chapter 00 and progress sequentially
2. **Read actively** – Each chapter includes examples, code snippets, and "Gotchas" to watch for
3. **Practice hands-on** – Complete the exercises in the `exercises/` folder
4. **Test yourself** – Take all 4 practice exams; aim for 80%+ before the real exam
5. **Review weak areas** – Use exam results to identify topics needing more study

## 📊 Exam Overview

- **Exam Name:** Claude Certified Architect
- **Format:** Multiple choice and scenario-based questions
- **Duration:** ~90 minutes
- **Passing Score:** ~70-80%
- **Topics Covered:** Claude fundamentals, API development, agent patterns, MCP, architecture best practices
